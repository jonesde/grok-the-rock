# grok-the-rock

Grok the Rock — stories with morals and recursive memes for children of all ages.

## Companions

- *Grok the Rock: Three Quiet Stories* — near hand (small, still, mending)
- *Hai Ikthiss: Three Tall Tales* — far hand (tall, pouring, far sight)
- *The Potter* (`ThePotter.md`) — process tale whose two hands are those books

## Lattice seal

The dual books share a 32-unit structure. The machine witness:

```bash
python3 scripts/verify_lattice.py
python3 scripts/verify_markers.py
```

See `Lattice.md` for stone / water / potter design notes. Short seal appears on the title-page credits when the lattice holds.
